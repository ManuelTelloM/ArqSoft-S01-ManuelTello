# Actividad 2 - Catalogo de videojuegos
# Manuel Jesus Tello May.

Para esta actividad se realizo por medio de visual studio una app que funciona como un catalogo de videojuegos.

Se utilizo el .NET para realizar el programa y se utilizo tambien css y javaScript para ponerle un diseño y cambiarle los colores a la app.



# Catalogo.
Permite agrefar titulos de videojuegos. Al agregar un video juego se tendran que ingresar unos datos para identificar al juego.
como por ejemplo: Nombre, tipo, consola y año que salio.
